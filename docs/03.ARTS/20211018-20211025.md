### 本地化有了新提升

### Location 按钮

可以方便的用于位置信息的申请。

```swift
LocationButton(.currentLocation) {  
// Fetch location with Core Location.
}.symbolVariant(.fill)
.labelStyle(.titleAndIcon)
```

### Await

Swift 5.5 新增了 `await` 属性，在使用 `async` 的方法里，可以用该字段实现和其他方法类似的逻辑。
