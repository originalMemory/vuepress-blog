## Tip

Swift `Codable` 使用时，属性默认值不能使用 `xx: Bool = false` 这种写法，是无效的。如果 json 数据里没有该字段，会导致反序列化失败。一种做法是包一下，代码如下：

```swift
class A: Codable {
    var b: Bool?
    var resolvedB: Bool {
        b ?? false
    }
}
```

还有其他更好的解决方案，不过准备工作比较麻烦，后面再整理。
